How the Algorithms Work

BFS (Breadth-First Search): Explores the graph level by level. It visits all direct friends first, then all friends-of-friends, and so on. Like ripples in a pond!

DFS (Depth-First Search): Explores deeply through one chain of friendships before backtracking. It follows one path as far as possible before trying another path.
